General

Welcome to x2d - crossplatform 2d game engine. The engine is built upon simple concepts such as Objects, Scenes, Viewports and Cameras. XML is chosen for configuration of everything in the engine.

x2d supports Physics, Bitmap fonts, Zones and Triggers, Music and Sound effects, Lua scripting, Particle systems and a lot more.

A plugin (compile-time) system is available to code extensions and add functionality to the XML configuration parsers.

x2d relies on Boost and glm libraries.
